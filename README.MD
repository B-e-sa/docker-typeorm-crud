## Introduction

An note/todo application using: 

- Node
- Express
- Cors
- React
- Typeorm 
- Docker 
- Postgres

## Installation
`` npm install ``<br>

### To start db:
(do this once (docker desktop needs to be open))<br>
`` npm run db ``<br>

### To start Back-End:
`` npm run serve`` <br>

### To start Front-End: <br>
`` npm run dev ``<br>
go to: ``http://localhost:5173/`` (default vite port)
