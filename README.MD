## Introduction

An note/todo application using: 

- Node
- React
- Typeorm 
- Docker 
- Postgres

## Installation
`` npm install ``<br>

### To start Front-End: <br>
`` npm run dev ``<br>
go to<br>
``http://localhost:5173/`` (default vite port)

### To start db:<br>
`` npm run db ``<br>

### To start Back-End: <br>
`` npm run serve`` <br>
